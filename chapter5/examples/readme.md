git 常用命令

```shell
git status — 查看本地代码的变化。
git diff — 查看具体的修改内容。
git add . — 将修改添加到暂存区。
git commit -m "message" — 提交修改。
git push origin <branch-name> — 推送修改到远程仓库。

```
